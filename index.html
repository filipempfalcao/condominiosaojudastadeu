Instruções para Atualização Mensal

Para manter o BI financeiro atualizado com novos relatórios mensais:

  1. Acesse seu repositório GitHub
  2. Edite o arquivo index.html
  3. Localize a seção "DADOS FINANCEIROS" no código
  4. Adicione os novos dados do mês no formato adequado
  5. Atualize a lista de meses no seletor
  6. Salve as alterações

O sistema atualizará automaticamente os gráficos e análises com os novos dados.
